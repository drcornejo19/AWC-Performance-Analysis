# 📊 AdventureWorks – Financial & Operational Performance Analysis

## 🏢 Business Context
AdventureWorks is a global retail company requiring financial and operational performance visibility to support strategic decision-making.

## 🎯 Objective
Develop an executive-level dashboard to monitor revenue, costs, profitability and key operational indicators, with focus on the United States market.

## 📈 Analytical Scope
- Revenue and COGS analysis
- Gross and Net Profit evaluation
- Margin tracking
- Country and state-level performance
- Time-based trend analysis
- KPI monitoring and variance analysis

## 🏗 Data Modeling
- Star schema design
- Fact and dimension tables
- DAX measure development
- Time intelligence calculations

## 📊 Dashboard Features
- Interactive filters
- Executive KPIs
- Geographic visualization
- Profitability breakdown
- Comparative period analysis

## 🛠 Technologies
- Power BI
- DAX
- Data Modeling
- Business Intelligence principles

## 💡 Business Impact
This dashboard provides decision-makers with clear financial visibility, enabling data-driven strategy, operational optimization and profitability analysis.
